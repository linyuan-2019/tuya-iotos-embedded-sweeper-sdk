# Tuya 嵌入式扫地机 SDK

[English](README.md) | [中文版](README_cn.md)

## 介绍

涂鸦扫地机 SDK，借助设备的联网能力，直接与涂鸦 IoT 平台，涂鸦 APP，建立通信链路并进行涂鸦标准数据交互的软件中间件，你可以用此SDK开发接入涂鸦云的扫地机产品。
