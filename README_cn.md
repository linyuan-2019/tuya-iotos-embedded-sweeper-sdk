# Tuya 嵌入式扫地机 SDK

[English](README.md) | [中文版](README_cn.md)

##介绍
涂鸦扫地机SDK是指在涂鸦IoTOS系统中针对基于Linux系统开发扫地机设备推出的软件开发包。提供标准C语言静态库头文件以及对应的说明文档。
